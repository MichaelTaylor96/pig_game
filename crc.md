## Game

### Responsibilities:
    - Switching turns
    - Picking a player to go first
    - Checking if a player has won
    - Starting / ending the game

### Collaborators:
    - Computer player
    - Human player


## Human player

### Responsibilities:
    - Keeps track of own score
    - Asks if human wants to roll / hold
    - Rolls a die for player
    - Ends turn

### Colaborators:
    - Game


## Computer player

### Responsibilities:
    - Keeps track of own score
    - Rolls / holds based on own strategy
    - Ends turn

### Collaborators:
    - Game
